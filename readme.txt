It is a HTTP api for getting and listing informations about pokemons, types and moves.

Endpoints
---------

get/pokemon
get/type
get/move
You can call the endpoints that above with name or type filters.

list/pokemons
list/types
list/moves
You can call the endpoints that above and sort them with sortby parameter.

You can sort the pokemons by base attack, base stamina, base defense
You can sort the moves by damage, energy, dps, duration
